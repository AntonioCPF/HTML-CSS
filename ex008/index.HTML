Principais formatações

Negrito/ Destaque

Nesta frase temos um termo em negrito usando atag <b>

Nesta frase temos um termo em destaque usando a tag <strong>

Italico/ Ênfase

Nesta frase temos um termo em italico utilizando a tag <i>

Nesta frase temos um termo em ênfase usando a tag <em>

Dica: voçe pode envelopar textos selecionados dentro de tags utilizando os comando ctrl+shift+p

Texto marcado

Podemos usar um texto marcado usando a tag <mark>

e no outro pagrafo eu tenho outro texto marcado

Estamos criando um texto grande e um texto pequeno usando as tags <big> e <small>,

texto deletado

Podemos marcar um texto como excluído para indicar que ele deveria ser lido mas não considerado com a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado com a tag <ins>

Existe também o sublinhado com a tag <u> (não semântica)

Texto sobre escrito

Para inserir coisas do tipo x20 + 3 utilizasse a tag <sup>

Texto subscrito

Para inserir coisas do tipo h2o utilizasse a tag <sub>
